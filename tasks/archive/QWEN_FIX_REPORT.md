## Phase 2: Add `updateProjectReport` function

Add this function to `src/index.ts` (near the other summary functions, before `finalizeTaskSummary`):

```typescript
function updateProjectReport(): void {
  const cfg = loadConfig();
  const reportPath = path.join(cfg.projectDir, 'PROJECT_REPORT.md');
  const all = loadSavings();
  const cum = getCumulativeSavings();
  const now = new Date().toISOString();
  
  const lines = [
    '# 📊 AutoClaude Project Report',
    '',
    `> Updated: ${now}  |  Tasks: ${cum.tasks}  |  Tokens Saved: ~${cum.tokensSaved.toLocaleString()}  |  Cost Saved: $${cum.costSaved.toFixed(2)}`,
    '',
    '---',
    '',
    '## 👥 Who Does What',
    '',
    '| 🧠 Role | 💻 System | 📝 Responsibility | Avg Tokens/Task |',
    '|---------|----------|-------------------|----------------|',
    '| **Planner** | Claude Code | Strategy, architecture, task writing, verification | ~7,000 |',
    '| **Dispatcher** | AutoClaude | Validation, dispatch, notifications, cost tracking | 0 (local) |',
    '| **Executor** | AI Agent | File ops, git, builds, deployments | ~25,000 |',
    '',
    '---',
    '',
    '## 📊 Cumulative Savings',
    '',
    '| Metric | Value |',
    '|--------|-------|',
    `| **Total Tasks** | ${cum.tasks} |`,
    `| **Total Tokens Saved** | **~${cum.tokensSaved.toLocaleString()}** |`,
    `| **Total Cost Saved (Opus 4.7)** | **$${cum.costSaved.toFixed(2)}** |`,
    `| **Average Savings/Task** | ~$${cum.tasks > 0 ? (cum.costSaved / cum.tasks).toFixed(3) : '0.00'} |`,
    '',
    '---',
    '',
  ];
  
  if (all.length > 0) {
    lines.push('## 📋 Task History');
    lines.push('');
    lines.push('| # | Task | Date | Claude | Agent | Saved |');
    lines.push('|---|------|------|--------|-------|-------|');
    for (const [i, s] of all.entries()) {
      const date = new Date(s.timestamp).toLocaleString('zh-CN', { month: 'short', day: 'numeric', hour: '2-digit', minute: '2-digit' });
      const name = s.taskName.replace(/^QWEN_/, '').replace(/_/g, ' ').substring(0, 25);
      const c = (s.claudeTokensIn + s.claudeTokensOut);
      const e = (s.estimatedExecutionTokensIn + s.estimatedExecutionTokensOut);
      lines.push(`| ${i + 1} | ${name} | ${date} | ~${c.toLocaleString()} | ~${e.toLocaleString()} | **~${s.tokensSaved.toLocaleString()}** |`);
    }
    lines.push('');
  }
  
  lines.push('---');
  lines.push('');
  lines.push('*Generated by AutoClaude v5.3 — get_project_report in Claude to view.*');
  
  fs.writeFileSync(reportPath, lines.join('\n'), 'utf-8');
}
```
