How It Works

Claude plans the strategy. AutoClaude dispatches tasks. Qwen Code or Cursor executes them — all in the background, all automatic.

psychology

1. Claude Plans

Claude Code analyzes the problem, designs the architecture, and writes a detailed task file with step-by-step instructions.

send

2. AutoClaude Dispatches

With one MCP tool call, AutoClaude pipes the task to Qwen Code or copies it to Cursor. Headless. YOLO. Instant.

terminal

3. Agents Execute

Qwen Code or Cursor runs the task independently using their own tokens. Claude is free for the next planning session.

Getting Started

Three commands to connect Claude Code to Qwen Code and Cursor.

# 1. Clone and install
git clone https://github.com/zhewenzhang/qwen-bridge.git
cd qwen-bridge && npm install && npm run build
# 2. Add to Claude Code settings
// In ~/.claude/settings.json:
"mcpServers": {
  "autoclaude": {
    "command": "node",
    "args": ["D:\\qwen-bridge\\dist\\index.js"]
  }
}
# 3. Restart Claude Code and dispatch your first task
Claude: Write QWEN_MY_TASK.md, then dispatch_to_qwen("QWEN_MY_TASK.md")
